CSS Types Used:
1. Inline CSS:
   - Applied to the <section> for background color:
     <section style="background-color: #f0f8ff";>
2. Internal CSS:
   - Inside <style> tag in the <head> for basic h1/h2 styling and hover effect on links.
3. External CSS:
   - Linked using <link rel="stylesheet" href="style.css">
   - Contains most of the styling for layout, colors, fonts, and effects.

CSS Selectors Used:
1. ID Selector:
   - #profile-pic used to style the profile image.
2. Class Selector:
   - .bio for the bio paragraph.
   - .card for the profile card container with box-shadow.
3. Element Selector:
   - Used for styling body, p, h1.
4. Group Selector:
   - h1, h2 used together in internal CSS.
5. Descendant Selector:
   - .contact p used to style <p> inside the contact section.
6. Attribute Selector:
   - a[href] used to style all links with an href attribute.
